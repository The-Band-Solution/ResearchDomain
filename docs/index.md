# ResearchDomain

`research_domain` e uma biblioteca Python para modelagem e gerenciamento de dados academicos de pesquisa.

Este site publica a documentacao tecnica do projeto e organiza o material que ja existe no repositorio em uma navegacao pronta para GitHub Pages.

## O que voce encontra aqui

- visao geral da documentacao do projeto
- requisitos e especificacoes funcionais
- descricao de arquitetura e desenho de software
- referencia da camada `domain`
- notas historicas sobre o modelo de orientacoes
